# FPGA Traffic Light Controller

## Project Overview
This project implements a Traffic Light Controller using Verilog HDL. The controller manages Red, Yellow, and Green traffic signals using a Finite State Machine (FSM).

## Features
- Verilog HDL implementation
- FSM-based traffic light control
- Red, Yellow, and Green signal sequencing
- Testbench for simulation
- Waveform verification

## Tools Used
- EDA Playground
- Verilog HDL
- EPWave

## Project Files
- traffic_light_controller.v - Main Verilog module
- traffic_light_tb.v - Testbench for simulation
- waveform.png - Simulation waveform screenshot

## Working
The controller cycles through the following states:

1. Green Light ON
2. Yellow Light ON
3. Red Light ON
4. Repeat

The state transitions are controlled by a clock signal and reset input.

## Applications
- Traffic Signal Control Systems
- FPGA Learning Projects
- Digital Design and FSM Implementation
